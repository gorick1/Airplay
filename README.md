# Alexa AirPlay Bridge - Home Assistant Addon Repository

Official Home Assistant addon repository for the Amazon Echo to AirPlay bridge.

## About

This repository contains a Home Assistant addon that streams Apple Music to Amazon Echo devices via virtual AirPlay receivers with full playback control.

## Installation

1. Add this repository to Home Assistant:
   - Settings ‚Üí Add-ons ‚Üí Create New Home Assistant Community Add-ons instance
   - Add the repository URL: `https://github.com/gorick1/Airplay`

2. Install the "Amazon Echo AirPlay Bridge" addon

3. Configure with Amazon OAuth credentials

4. Start the addon
